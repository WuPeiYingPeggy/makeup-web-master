美妝網站(團體製作)
------------------
使用 react 做前端框架渲染頁面，用 node.js express 做伺服器連接 mysql ，並以 fetch 來做前後端資料連接

-----------------------
需事前安裝 node.js

安裝方式：分別在makeup-web資料夾和server資料夾使用終端機個別輸入npm install完成安裝對應的nodejs

sql使用mysql連接，使用軟體xampp
